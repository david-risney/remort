# Specs

This directory holds feature specifications following the [Spec Kit](https://github.com/github/spec-kit) workflow.

Each feature gets a numbered directory:

```
specs/
├── 001-feature-name/
│   ├── spec.md          # User-focused specification (WHAT/WHY)
│   ├── plan.md          # Technical implementation plan (HOW)
│   ├── tasks.md         # Executable task list
│   └── checklists/      # Quality validation checklists
└── 002-next-feature/
    └── ...
```

See [DEVELOPMENT.md](../DEVELOPMENT.md) for the full Spec Kit workflow.
